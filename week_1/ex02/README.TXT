Exercise 03: ft_strlen
Exercise name: ft_strlen

Turn-in directory: ex03/

Files to turn in: ft_strlen.c

Allowed functions: None

Description: Reproduce the behavior of the standard strlen function. It must count the number of characters in a string, excluding the terminating null byte ('\0').

Constraints: You are not allowed to use array indexing (e.g., str[i]). You must use pointer arithmetic to traverse the string.

Function declaration:

int ft_strlen(char *str);
