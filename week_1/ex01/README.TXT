Exercise 01: ft_ultimate_div_mod
Exercise name: ft_ultimate_div_mod

Turn-in directory: ex01/

Files to turn in: ft_ultimate_div_mod.c

Allowed functions: None

Description: Write a function that takes two pointers to integers (a and b). It must divide the value of a by b. The quotient shall be stored in a and the remainder in b.

Constraints: Strict use of dereferencing is required.

Function declaration:

void ft_ultimate_div_mod(int *a, int *b);
