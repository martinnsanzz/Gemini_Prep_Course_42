Exercise 05: ft_range
Exercise name: ft_range

Turn-in directory: ex05/

Files to turn in: ft_range.c

Allowed functions: malloc

Description: Create a function that returns an array of integers containing all values between min (inclusive) and max (exclusive).

Constraints: If min >= max, the function must return NULL. You must calculate the exact amount of memory needed.

Function declaration:

int *ft_range(int min, int max);
